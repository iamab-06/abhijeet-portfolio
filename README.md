# Abhijeet Bhat | Software Engineering Portfolio

Welcome to my personal portfolio repository! This is a modern, full-stack web application designed to showcase my engineering projects, technical skills, and background in Computer Science and Engineering.

## 🚀 Tech Stack

- **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)
- **Library:** [React](https://react.dev/)
- **Language:** TypeScript & JavaScript
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **3D Graphics:** [Three.js](https://threejs.org/) via `@react-three/fiber`

## ✨ Features

- **Interactive 3D Backgrounds:** Dynamic, performant starfield scenes using Three.js.
- **Modern UI/UX:** Clean aesthetics built with Tailwind CSS, featuring glassmorphism and gradient highlights.
- **Fluid Animations:** Scroll-triggered entry animations using Framer Motion.
- **Centralized Data Layer:** Scalable architecture separating content from UI components.
- **Fully Responsive:** Seamless experience across desktop, tablet, and mobile devices.

## 💻 Local Setup Instructions

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/iamab-06/abhijeet-portfolio.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd abhijeet-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.
